
# 📉 Customer Churn Analysis

---

## 📌 Problem Statement

Customer churn is a major challenge for businesses.  
The goal of this project is to analyze customer data and identify factors that lead to churn, helping businesses improve retention strategies.

---

## 🎯 Objectives

- Identify key factors affecting customer churn  
- Analyze customer behavior patterns  
- Provide actionable insights to reduce churn  
- Support data-driven decision making  

---

## 📂 Dataset

- Dataset: Telco Customer Churn Dataset  
- Source : [https://www.kaggle.com/datasets/blastchar/telco-customer-churn]
- Features include:
  - CustomerID  
  - Gender  
  - Tenure  
  - Contract Type  
  - Monthly Charges  
  - Total Charges  
  - Churn (Target Variable)  

---

## 🛠️ Tools & Technologies

- Python  
  - Pandas  
  - NumPy  
  - Matplotlib / Seaborn  
- SQL  
- Power BI / Excel (for dashboard)

---

## 🔍 Steps Performed

1. Data Cleaning  
   - Handled missing values  
   - Converted data types  

2. Data Preprocessing  
   - Feature selection  
   - Encoding categorical variables  

3. Exploratory Data Analysis (EDA)  
   - Churn distribution  
   - Correlation analysis  
   - Feature-wise comparison  

4. Visualization  
   - Bar charts  
   - Pie charts  
   - Heatmaps  

5. Insight Generation  
   - Identified high-risk customer segments  

---

## 📊 Key Insights

- 📉 Customers with **month-to-month contracts** have higher churn  
- 💰 Higher **monthly charges** lead to higher churn probability  
- ⏳ Customers with **low tenure** are more likely to churn  
- 📡 Lack of **tech support / online security** increases churn
