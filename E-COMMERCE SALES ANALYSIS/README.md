

# 📊 E-Commerce Sales Analysis

## 📌 Project Overview

This project analyzes e-commerce sales data to uncover trends, customer behavior, and business insights.

## 🎯 Objectives

* Analyze sales performance
* Identify top-performing products
* Understand customer purchasing patterns
* Provide actionable business recommendations

## 🛠️ Tech Stack

* Python (Pandas, NumPy)
* SQL
* Excel
* Power BI

## 📂 Dataset

* Source: [https://www.kaggle.com/datasets/bravehart101/sample-supermarket-dataset]
* Size: [9995 rows]

## 🔍 Key Steps

1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis (EDA)
3. SQL Query Analysis
4. Dashboard Creation (Power BI)

## 📊 Key Insights

* Top 20% products generate 60% revenue
* Seasonal trends observed in Q4
* High churn in specific customer segments
